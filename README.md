# Admin Boilerplate

This is a Node.js application with TypeScript and Express.

## Features
- TypeScript
- Express
- Nodemon for development
- Environment variables support with dotenv
- ESLint for code linting
- Winston for logging

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run in development mode:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Run production build:
   ```bash
   npm start
   ```
